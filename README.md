# エクストリームプログラミングの肌触り

ここではエクストリームプログラミングについてのノウハウをまとめます。  

エクストリームプログラミングのプラクティスの項目は、[Scrum Starter Guide](https://www.ryuzee.com/wp-content/site_images/7150/scrum-starter-guide.pdf)を参考にしています。

### 共同のプラクティス

- イテレーション
- 共通の用語
- オープンなワークスペース
- ふりかえり

### 開発のプラクティス

- テスト駆動開発
- ペアプログラミング
- リファクタリング
- ソースコードの共同所有
- 継続的インテグレーション
- YAGNI

### 管理者のプラクティス

- 責任の受け入れ
- 援護
- 四半期ごとの見直し
- ミラー
- 最適なペース

### 顧客のプラクティス

- ストーリー
- リリース計画
- 受け入れテスト
- 短期リリース
